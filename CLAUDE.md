# Claude Instructions — ts-types

This project is an advanced TypeScript interview prep course for a frontend engineer with TS fundamentals.

## How each section works

1. **Teach concepts interactively in chat** — explain theory with examples, ask questions to check understanding, go back and forth until the user is solid
2. **Write a reference `.md` file** — once the section is done, write a doc capturing all the key concepts and examples
3. **Run exercises in chat** — give problems/challenges directly in the conversation to test understanding

## File naming

Reference docs use section numbers so they sort correctly, e.g. `01-mental-model.md`, `02-type-system-foundations.md`.

## Syllabus & progress tracking

See `README.md` — update checkboxes (`[ ]` → `[x]`) as each section is completed.
